# 🍽️ Crêpe Bouali - Blender Project

## 📌 Description

This is a 3D Blender project representing a mobile crêpe cart (“Crêpe Bouali”). The scene includes a detailed street food cart with textures, branding, utensils, and cooking elements.

The goal of this project is to simulate a realistic food cart environment with good lighting, materials, and composition.

## 🧱 Project Contents

* Crêpe cart model
* Wheels and structure
* Textured panels (branding + food images)
* Cooking tools (pans, cups, utensils)
* Props and environment objects
* Lighting setup

---

## 🛠️ Software Used

* Blender (3D modeling & rendering)

---

## 🚀 How to Use

1. Open Blender
2. Load the `.blend` file from the project folder
3. Go to **Rendered View** to see materials properly
4. Press `F12` to render the scene
